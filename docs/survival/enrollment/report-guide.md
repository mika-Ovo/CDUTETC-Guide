# 报到须知

## 报到时间

具体报到时间请以录取通知书为准，一般在8月底至9月初。

## 报到流程

### 线上报到

::: warning 重点提示
以学校发放的新生入学手册（和录取通知书一起邮寄到家）上有详细教程，请仔细阅读。
:::

1. **登录迎新系统**：使用学号和密码登录学校迎新系统
2. **信息核对**：核对个人信息，如有错误及时联系教务处
3. **上传材料**：按要求上传身份证、证件照等材料扫描件
4. **缴费确认**：确认学费、住宿费等费用缴纳情况
5. **宿舍预选**：在规定时间内选择宿舍（如开放此功能）

### 线下报到

1. **到校登记**：到学校指定地点进行身份验证
2. **领取材料**：领取校园一卡通、学生证、新生手册等
3. **宿舍入住**：凭相关凭证到宿舍办理入住手续
4. **体检安排**：按学校安排进行新生体检
5. **学院报到**：到所在学院完成注册手续
6. **领取军训物资**：按通知时间领取军训服装等物资

## 需要携带的材料

- 录取通知书
- 身份证及复印件
- 证件照（一寸、二寸）

> **注意**：需要申请贫困助学金的同学需要提前在家乡准备好对应材料（如贫困证明等），具体要求请参考新生入学手册或咨询学院辅导员。

## 注意事项

- 提前关注学校官方通知，了解最新报到要求
- 注意交通安全，保管好个人财物
- 按时报到，如有特殊情况需提前请假
- 报到期间校园人流量大，注意防暑降温
