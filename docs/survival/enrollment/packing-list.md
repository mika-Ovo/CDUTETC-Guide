# 必备物品清单

## 报道需求

- 录取通知书
- 身份证及复印件（多准备几份）
- 证件照（一寸、二寸，蓝底白底各准备一些）
- 户口本复印件（部分学校需要）
- 高中档案（按录取通知书要求）
- 党团组织关系证明
- 费用相关（银行卡、现金少量）

## 床上用品

- 床垫（推荐网购到校）
- 枕头
- 铺盖（棉被或羽绒被）
- 凉席
- 床帘（保护隐私必备）
- 床单、被套、枕套（两套换洗）
- 蚊帐（夏季必备）

## 生活用品

- 洗脸盆
- 桶
- 个人洗护用品（推荐到校后超市购买）
- 毛巾（多条）
- 牙刷、牙膏、漱口杯
- 洗发水、沐浴露、护发素
- 洗衣液/洗衣粉、肥皂盒
- 衣架、夹子
- 吹风机
- 拖鞋（室内、室外各一双）
- 雨伞/雨衣
- 保温杯

## 学习用品

- 笔记本电脑（根据专业需求准备）
- 充电器、数据线
- 插排（多插孔，安全认证）
- 台灯（护眼款）
- 书包
- 笔记本、笔、文件夹
- 书籍（专业相关或课外读物）

## 电子产品

- 手机及充电器
- 耳机（有线/无线）
- U盘（存储重要文件）
- 移动硬盘（可选）
- 相机（可选，记录校园生活）

## 宿舍实用物品

- 收纳盒、收纳袋（整理衣物和杂物）
- 粘钩（挂东西）
- 垃圾桶及垃圾袋
- 锁（衣柜、抽屉用）
- 小药箱（常用药品：感冒药、创可贴、肠胃药等）
- 针线包
- 多功能螺丝刀

## 其他建议

- 适量现金（刚到校可能用得上）
- 常用证件电子版备份（存手机或云盘）
- 衣物（根据季节准备，学校周边可购买）
- 鞋子（运动鞋、皮鞋等）
